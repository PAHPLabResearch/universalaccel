# universalaccel

**universalaccel (UA)** is a unified R pipeline for computing harmonized accelerometer summary metrics from raw data across ActiGraph, Axivity, and GENEActiv devices. 
It provides stable, reproducible epoch-level outputs and a downstream analysis workflow for daily intensity distributions interpretation using NHANES (National Health and Nutrition Examination Survey) references.
